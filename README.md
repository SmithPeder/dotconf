# iOS dotfiles

### Currently setting up
First get `brew`

### iTerm2

- brew install iTerm2
- NORD iTerm2 profile

### ZSH

```zsh
// Get zsh
$ brew install zsh
$ brew install zsh-completions
// Get oh-my-zsh
$ sh -c "$(curl -fsSL https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"
// Then make and move .oh-my-zsh to the plugins folder
$
```
